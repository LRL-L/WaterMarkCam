# WaterMarkCam 水印相机使用说明

> 版本 v1.0.4 | 支持 iOS 和 Android 设备

---

## 📱 快速开始

### 访问应用
在手机浏览器中打开：
```
https://lrl-l.github.io/WaterMarkCam/
```

---

## 🍎 iOS 使用指南

### 系统要求
- **iOS 11.3+**（建议 iOS 13+）
- **Safari 浏览器**（推荐）

### 第一次使用

#### 1. 打开应用
- 使用 **Safari** 浏览器访问网址
- 首次访问会提示"允许访问摄像头"，点击 **"允许"**

#### 2. 安装为桌面应用（可选但推荐）
1. 点击底部工具栏的 **"分享"图标** 📤
2. 向下滚动，找到 **"添加到主屏幕"**
3. 点击右上角的 **"添加"**
4. 应用图标会出现在主屏幕上

#### 3. 启动相机
- 点击 **"启动相机"** 按钮
- 如果提示权限请求，选择 **"允许"**
- 相机会自动切换到后置摄像头

### 功能使用

#### 📷 拍照
1. 点击屏幕下方的 **大圆形按钮**（拍照键）
2. 听到快门声，照片拍摄完成
3. 自动跳转到预览界面

#### 🔍 扫描二维码
1. 点击右下角的 **二维码图标** 📱
2. 将二维码对准屏幕中央的扫描框
3. 识别成功后会自动填入二维码内容
4. 点击 **"取消"** 返回拍照模式

#### 💾 保存照片
**方法 A：分享到相册（推荐）**
1. 拍照后进入预览界面
2. 点击 **"储存至相簿"** 按钮
3. 在弹出的分享菜单中选择 **"存储图像"**
4. 照片会保存到系统相册

**方法 B：下载到文件**
1. 点击 **"下載照片"** 按钮
2. 照片会保存到 Safari 的"下载"文件夹
3. 在"文件" App 中可以找到

#### 🔄 重拍
点击 **"重拍"** 按钮返回拍照模式

### iOS 特别注意事项

⚠️ **摄像头权限**
- 如果相机无法启动，请检查：
  - 设置 → Safari → 摄像头 → 选择"允许"
  
⚠️ **私密浏览模式**
- 私密浏览模式下可能无法访问摄像头
- 建议使用普通模式

⚠️ **离线功能**
- 首次访问后会自动缓存
- 之后即使没有网络也能正常使用
- 系统可能会定期清理缓存，建议收藏网址

## 🤖 Android 使用指南

### 系统要求
- **Android 5.0+**（建议 Android 8.0+）
- **Chrome 浏览器**（推荐）

### 第一次使用

#### 1. 打开应用
- 使用 **Chrome** 浏览器访问网址
- 首次访问会提示"允许访问摄像头"，点击 **"允许"**

#### 2. 安装为应用（可选但推荐）
**方法 A：Chrome 提示安装**
1. 访问网址后，地址栏会显示"安装"图标 ⊕
2. 点击图标，弹出"添加到主屏幕"对话框
3. 点击 **"添加"** 或 **"安装"**
4. 应用图标会出现在桌面

**方法 B：手动添加**
1. 点击浏览器右上角 **⋮（菜单）**
2. 选择 **"添加到主屏幕"** 或 **"安装应用"**
3. 输入应用名称（默认"水印相机"）
4. 点击 **"添加"**

#### 3. 启动相机
- 点击 **"启动相机"** 按钮
- 如果提示权限请求，选择 **"允许"**
- 相机会自动切换到后置摄像头

### 功能使用

#### 📷 拍照
1. 点击屏幕下方的 **大圆形按钮**（拍照键）
2. 听到快门声，照片拍摄完成
3. 自动跳转到预览界面

#### 🔍 扫描二维码
1. 点击右下角的 **二维码图标** 📱
2. 将二维码对准屏幕中央的扫描框
3. 识别成功后会自动填入二维码内容
4. 点击 **"取消"** 返回拍照模式

#### 💾 保存照片
1. 拍照后进入预览界面
2. 点击 **"下載照片"** 按钮
3. 照片会保存到 **Download（下载）** 文件夹
4. 可以在相册或文件管理器中查看

#### 🔄 重拍
点击 **"重拍"** 按钮返回拍照模式

### Android 特别注意事项

⚠️ **摄像头权限**
- 如果相机无法启动，请检查：
  - 设置 → 应用 → Chrome → 权限 → 相机 → 启用

⚠️ **存储权限**
- 保存照片可能需要存储权限
- Chrome 会自动申请，点击"允许"即可

⚠️ **后台运行**
- Android 系统可能会清理后台应用
- 如需长时间使用，建议锁定应用到后台

⚠️ **离线功能**
- 首次访问后会自动缓存
- 之后即使没有网络也能正常使用
- 定期联网可确保使用最新版本

## ✨ 高级功能

### 🔄 切换前后摄像头
- 点击左下角的 **切换图标** 🔄
- 可在前置和后置摄像头之间切换
- 自拍时使用前置摄像头

### 📐 水印设置
- 水印会自动显示在照片右下角
- 包含：
  - ✅ 二维码内容
  - ✅ 拍摄时间（年-月-日 时:分:秒）
- 水印透明度为 90%，不影响照片清晰度

### 📱 PWA 离线功能
- 第一次访问时会自动缓存所有资源
- 之后可以在**完全没有网络**的情况下使用
- 所有功能（拍照、扫码、保存）均可离线使用

## 🆘 常见问题

### Q1: 摄像头显示黑屏怎么办？
**解决方法：**
1. 检查浏览器是否有摄像头权限
2. 退出应用后重新进入
3. 重启浏览器
4. 检查是否有其他应用占用摄像头

### Q2: 无法保存照片到相册（iOS）
**原因：** iOS 的安全机制限制
**解决方法：**
- 使用"储存至相簿"按钮，然后选择"存储图像"
- 或者下载后在"文件" App 中手动移动到相册

### Q3: 二维码扫描识别不出来
**改进识别率：**
1. 确保光线充足
2. 将二维码完整放入扫描框
3. 保持手机稳定，避免抖动
4. 调整距离（约 10-30cm 最佳）
5. 确保二维码清晰无污损

### Q4: 照片保存在哪里？
**iOS：**
- "储存至相簿" → 系统相册
- "下載照片" → Safari 下载文件夹（文件 App 可查看）

**Android：**
- "下載照片" → Download 文件夹
- 可在相册或文件管理器中查看

### Q5: 离线能用吗？
**回答：** 能！
- 第一次访问时需要联网（自动缓存）
- 之后完全断网也能正常使用
- 所有功能（拍照、扫码、保存）都支持离线

### Q6: 应用图标消失了怎么办？
**iOS：**
- 系统可能清理了缓存
- 重新访问网址并添加到主屏幕

**Android：**
- 在应用抽屉中查找"水印相机"
- 或重新安装 PWA

### Q7: 可以编辑二维码内容吗？
**当前版本：** 不支持手动编辑
- 只能通过扫描二维码自动识别
- 未来版本会考虑添加手动输入功能

### Q8: 水印可以自定义吗？
**当前版本：** 固定样式
- 包含二维码和时间戳
- 位置在照片右下角
- 未来版本会考虑添加自定义选项

## 📝 使用技巧

### 💡 提高拍照质量
1. **光线充足**：在明亮环境下拍摄
2. **稳定手机**：避免抖动模糊
3. **清洁镜头**：定期擦拭摄像头
4. **调整距离**：根据拍摄对象调整距离

### 💡 快速扫码技巧
1. **提前准备**：先准备好二维码
2. **光线充足**：扫码需要好的光线
3. **平整展示**：确保二维码平整无折痕
4. **适当距离**：10-30cm 识别率最高

### 💡 离线使用场景
- 🏞️ 户外无信号区域（山区、景区）
- ✈️ 飞行模式下使用
- 🌐 网络受限环境
- 💰 节省移动数据流量

## 🔒 隐私保护

### 数据安全承诺
✅ **所有数据本地处理**
- 照片仅保存在您的设备上
- 不会上传到任何服务器
- 二维码识别完全在本地完成

✅ **不收集个人信息**
- 不记录您的拍摄内容
- 不追踪您的使用行为
- 不要求登录或注册

✅ **摄像头权限**
- 仅在您使用时才访问摄像头
- 关闭应用后自动释放权限
- 您可以随时在系统设置中撤销权限
